Digital Wallet App Design

This project focuses on designing a digital wallet interface tailored for students — starting from wireframes to high-fidelity layout. The app allows users to check balances, scan QR codes for payment, view transaction history, and navigate easily with a clean UI.

Key Features

  • Balance & transaction history view
  • QR code payment feature
  • Minimal and modern UI design

Objectives

  • Practice mobile UI design
  • Understand student-focused financial tools
  • Create a clean and structured layout
