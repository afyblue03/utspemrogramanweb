Illustration & Sketch Practice

This project contains digital drawing and sketching practice focused on improving visual observation, line control, and understanding body proportions. The drawing is created using a freehand sketching style with character reference to study poses, expression, and structure.

Key Features

  • Basic linework and shading exercises
  • Character pose study (Spiderman reference)
  • Experimentation with line weight and illustration style
  • Improving proportion understanding and character form

Objectives

  • Strengthen digital drawing skills
  • Develop consistent sketch flow
  • Boost creativity through freestyle illustration
  • Add more visual artwork to portfolio
