Personal Portfolio Website

This project showcases my personal portfolio website, containing my profile, interests, and a collection of projects I have worked on. The design focuses on clean typography, a simple grid structure, and responsive behavior to ensure smooth viewing on both mobile and desktop.

Key Features

  • Profile and Interests Section
  • Project List with Dedicated Detail Pages
  • Fully Responsive Layout

Objectives

  • Build a professional personal website
  • Improve layout structure and UI consistency
  • Provide a clean portfolio for academic or job needs
